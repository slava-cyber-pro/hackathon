BudgetSphere — a web-based financial tracking PWA for individuals and teams.

**Core features**: spending tracker with categories, income management, investment tracking, rich chart visualizations, personal/team budgets with limits, team collaboration with member filtering and role-based access.

**Tech stack**: Python FastAPI, PostgreSQL 16, Redis 7, React 18+ (TypeScript, Vite), Tailwind + shadcn/ui, Recharts, Docker Compose.

**Why:** Hackathon project starting 2026-03-30. User wants a full-stack production-quality app with modular architecture, comprehensive tests, PWA support (add-to-home-screen, standalone display), and mobile-first responsive design.

**How to apply:** All implementation decisions should reference PROJECT_OVERVIEW.md for feature/architecture details. Prioritize milestones in order. Maintain modular separation (services layer, versioned API, component-based frontend).
